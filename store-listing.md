# Chrome Web Store 등록 정보

## 확장 프로그램 이름
21Lab AI Agent

## 짧은 설명 (132자 이내)
네이버 블로그 AI 원고 생성 도구. 키워드 입력만으로 SEO 최적화된 블로그 글을 자동 생성합니다.

## 상세 설명
21Lab AI Agent는 네이버 블로그 콘텐츠 제작을 위한 AI 기반 원고 생성 도구입니다.

### 주요 기능
- AI 기반 블로그 원고 자동 생성
- 다양한 AI 모델 선택 (GPT-4, GPT-5, Solar 등)
- 참조 원고 기반 맞춤 콘텐츠 생성
- 원고 큐 관리 및 복사 기능
- 네이버 블로그 SEO 최적화

### 사용 방법
1. 확장 프로그램 아이콘 클릭하여 사이드 패널 열기
2. AI 모델 선택
3. 키워드 입력 (선택: 참조 원고 추가)
4. "원고 생성" 버튼 클릭
5. 생성된 원고 더블클릭으로 복사

## 준비해야 할 에셋

### 1. 스토어 아이콘 (필수)
- 크기: 128x128 PNG
- 현재 SVG 아이콘을 PNG로 변환 필요

### 2. 스크린샷 (필수, 최소 1장)
- 권장 크기: 1280x800 또는 640x400
- Side Panel이 열린 상태에서 캡처
- 네이버 블로그 페이지와 함께 캡처하면 좋음

### 3. 프로모션 이미지 (선택사항)
- Small tile: 440x280
- Large tile: 920x680
- Marquee: 1400x560

## 비공개 배포 설정

Chrome Web Store Developer Dashboard에서:
1. "항목 추가" 클릭
2. ZIP 파일 업로드
3. 스토어 등록 정보 입력
4. **공개 범위 → "비공개" 선택**
5. 제출

비공개 URL을 받으면 해당 링크로만 설치 가능
